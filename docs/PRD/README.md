# PRD 버전 관리

## 버전 정책

| 버전대 | 의미 |
|--------|------|
| `v0.x` | 초안 / 방향 정의 단계 — 아직 확정 전 |
| `v1.x` | 확정 / 개발 착수 기준 문서 |
| `v2.x` | 기능 추가 / 개선 반영 |

## 업데이트 방법

PRD를 수정할 때는 기존 파일을 직접 수정하지 않고
새 버전 파일을 만들어 이 목록에 추가합니다.

```
PRD_v0.1.md  ← 현재
PRD_v0.2.md  ← 다음 업데이트
PRD_v1.0.md  ← 확정본
```
