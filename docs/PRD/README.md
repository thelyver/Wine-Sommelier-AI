# PRD 버전 관리

## 버전 정책

| 버전대 | 의미 |
|--------|------|
| `v0.x` | 초안 / 방향 정의 단계 — 아직 확정 전 |
| `v1.x` | 확정 / 개발 착수 기준 문서 |
| `v2.x` | 기능 추가 / 개선 반영 |
